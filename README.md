# Gold Edge Business Solution — Agency Website

**Tagline:** Smart Websites That Grow Your Business
**Location:** Rampurhat, Birbhum, West Bengal (near Tarapith)

A clean, modern, fully responsive agency website built with vanilla HTML, CSS and JS.
Optimized for fast load and SEO — ready to deploy on Netlify.

## Files

| File | Purpose |
|------|---------|
| `index.html` | Main page — all sections (Hero, Services, Pricing, About, Why, Payment, FAQ, Contact, Footer) |
| `style.css`  | Premium minimal theme with light/dark mode |
| `script.js`  | Theme toggle, mobile nav, scroll reveal, contact form |
| `robots.txt` | Search engine crawl rules |
| `sitemap.xml`| Sitemap for SEO |
| `netlify.toml` | Netlify build & header config |

## Deploy on Netlify (drag & drop)

1. Visit https://app.netlify.com/drop
2. Drag the unzipped `goldedge` folder onto the page
3. Done — your site is live.

## Customize

- Phone numbers: search `+919999999999` and `+91 99999 99999` in `index.html`
- Email: `hello@goldedge.in`
- Map: Google Maps embed already targets Rampurhat — change in `index.html` if needed
- Colors: edit CSS variables at the top of `style.css`
